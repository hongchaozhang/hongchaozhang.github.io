iOS代码查重工具

很多IDE里面都有自己的代码查重工具，比如WebStorm的代码查重工具做的非常好用。但是Xcode里面至今没有自己的代码查重工具。这里调研了一些常见的代码查重工具，最后选择PMD里面的CPD，并将其集成到Xcode中，重复代码会以warning的形式出现在Xcode里面。

jscpd

jscpd

Supported languages：

  • JavaScript
  • Java
  • YAML
  • CoffeeScript
  • C++
  • Haxe
  • PHP
  • C#
  • TypeScript
  • Go
  • Python
  • Mixed HTML
  • Ruby
  • C
  • SCSS
  • Less
  • CSS
  • erlang
  • Swift
  • xml/xslt
  • Objective-C
  • Puppet
  • Twig
  • Vue.js

优点

  • 安装使用简单
  • 运行速度快。

缺点：

  • 没有GUI
  • 算法简单：复制一段，修改其中的一行，就不会被认为复制。
