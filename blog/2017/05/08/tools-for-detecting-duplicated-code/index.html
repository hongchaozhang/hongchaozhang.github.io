jscpd

jscpd

Supported languages：

JavaScript Java YAML CoffeeScript C++ Haxe PHP C# TypeScript Go Python Mixed HTML Ruby C SCSS Less CSS erlang Swift xml/xslt Objective-C Puppet Twig Vue.js

优点

  • 安装使用简单
  • 运行速度快。

缺点：

  • 没有GUI
  • 复制1行，修改其中的一行，就不会被认为复制。

PMD

PMD

  • Eclipse plugin
  • NetBeans plugin
  • JBuilder plugin
  • JDeveloper plugin
  • IntelliJ IDEA plugin
